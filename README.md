# Air Pollution Prediction System

## About
A machine learning project that predicts the Air Quality Index (AQI) 
based on pollutant levels using Linear Regression.

## Technologies Used
- Python
- Scikit-learn (Linear Regression)
- Pandas
- NumPy

## How It Works
1. Takes pollutant values as input (PM2.5, PM10, NO2, SO2, CO)
2. Predicts the AQI using a trained Linear Regression model
3. Classifies air quality as Good / Moderate / Unhealthy / Hazardous

## How to Run
pip install pandas numpy scikit-learn
python air_pollution_prediction.py

## Model Performance
- Evaluated using RMSE and R² Score
- 80% training / 20% testing split
